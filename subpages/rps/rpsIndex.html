purpose and goal

I wanted to get familiar with DOM, and I decided to achieve that by building many projects.

general requirements

As always, apart from the bare minimum that the generated website must be responsive. I had a few requirements in mind to be able to call this project a success:

  • The resulting website must match the design on a Figma file, that I created for it
  • Add an animation that makes an image of what user/computer chose bigger

challenges

  1. Surprisingly, for me, I had some issues with scopes of variables. Variables that were defined outside of the function were not 'found' by my code editor, so at my first attempt, I nested all of my code inside of the one function. After I decided to rewrite my code this problem never surfaced again, so I am not completely sure what I did wrong the first time. However, I suspect that I must have simply misspelled variable names.

wrapping up

I learned CSS Grid (for the second time), got to practice DOM manipulation and learned how to make elements undraggable.

Overall, this assignment was super fun. As always, looking back at my projects, I tend to forget how many headaches each of them had caused me.
